# Company Overview
Elist is a global e-commerce company founded in 2018 that sells popular electronics through its website and mobile app. The company uses multiple marketing channels, including email, SEO, and affiliate links, and its top-selling brands include Apple, Samsung, and ThinkPad. This project analyzes Elist’s messy operational data across orders, customers, products, and geography to uncover insights that support better decision-making across finance, sales, product, and marketing.
